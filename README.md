## 🚀 Overview

**Zengi AML Platform** is a decentralized whistleblower network designed to combat money laundering activities. It allows users to anonymously submit reports, which are verified by a decentralized jury. Once verified, rewards are distributed to both the whistleblower and the jurors. Built on the Polygon blockchain, Zengi AML ensures transparency, privacy, and security through smart contracts and decentralized storage solutions like IPFS.

### Key Features

- **🕵️ Anonymous Reporting:** Submit reports without revealing your identity.
- **⚖️ Decentralized Verification:** Ensure fairness and transparency through a decentralized jury.
- **💸 Reward Distribution:** Earn rewards for valid reports and successful verifications.
- **🔒 Secure and Private:** Data is securely stored and transactions are protected by smart contracts.
- **⛓️ Blockchain Technology:** Powered by the Polygon network for low-cost and fast transactions.

## 🛠️ Tech Stack

- **Blockchain Platform:** Polygon (Ethereum Layer 2)
- **Smart Contracts:** Solidity, Hardhat
- **Frontend:** React.js, Ethers.js
- **Storage:** IPFS (InterPlanetary File System)
- **Node Infrastructure:** Alchemy

## 📦 Installation

To get started with the Zengi AML Platform, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nzengi/zengi_AML_platform.git
   cd zengi_AML_platform
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Compile smart contracts**:
   ```bash
   npx hardhat compile
   ```
